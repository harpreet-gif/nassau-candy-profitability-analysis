# 🍬 Product Line Profitability & Margin Performance Analysis for Nassau Candy Distributor

## 📌 Project Overview
This project focuses on analyzing **product profitability, gross margins, and division-level financial performance** for Nassau Candy Distributor using data analytics and business intelligence techniques.

The objective is to identify:
- High-profit products
- Low-margin risk products
- Revenue vs profit imbalance
- Cost-heavy products
- Profit concentration dependency

An interactive **Streamlit dashboard** was developed to provide real-time business insights and financial analytics.

---

## 🚀 Features
- Product profitability analysis
- Gross margin calculation
- Division-wise financial performance analysis
- High-sales low-margin product detection
- Pareto (80/20) profit concentration analysis
- Cost vs sales diagnostics
- Interactive Streamlit dashboard
- CSV report export
- PDF report generation
